## Installation Instructions

### Quick Install
1. Download `zierhofer-wordpress-theme-v3.0.zip` from GitHub
2. WordPress Admin → Appearance → Themes → Add New → Upload
3. Choose ZIP file and click "Install Now"
4. Click "Activate"
5. Configure in Appearance → Customize

### Detailed Instructions
See `INSTALLATION-GUIDE.md` for complete step-by-step instructions.
